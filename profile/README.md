# WitnessDevs

A collaborative space for Jehovah’s Witness developers who enjoy building clear, thoughtful, and open-source software.
Focused, minimal, and designed to make development feel simple.

## What We Create

* Lightweight tools and utilities
* Clean, consistent icon libraries
* Web components and UI experiments
* Projects that prioritize clarity and usability

## Our Approach

* Keep it simple
* Write maintainable code
* Communicate clearly
* Support each other
* Build things that make developers’ lives easier

## Join the Team

If you’re one of Jehovah’s Witnesses and enjoy coding, designing, or improving ideas, you’re welcome to collaborate.
Contributions can be small, large, or somewhere in between — every improvement matters.

## Current Projects

* **[Vector](https://github.com/WitnessDevs/Vector)** - A lightweight CSS framework for building responsive websites
* **[Vector Icons](https://github.com/WitnessDevs/icons)** — Official open source SVG icon library for Vector
